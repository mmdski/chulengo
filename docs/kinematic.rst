==============
Kinematic wave
==============

.. math:: q_t + f\left(q\right)_x = 0

.. math:: q_t + f\left(q\right)'q_x = 0

.. math:: q = h

.. math:: f\left(h\right) = CS^{1/2}_0h^{3/2}

.. math:: f\left(h\right)' = \frac{3}{2}CS_0^{1/2}h^{1/2}
