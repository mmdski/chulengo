==============
Kinematic wave
==============

.. math:: q_t + f\left(q\right)_x = 0

.. math:: q_t + f\left(q\right)'q_x = 0

.. math:: q = h

.. math:: f\left(h\right) = \frac{1}{n}h^{2/3}S^{1/2}_0

.. math:: f\left(h\right)' = \frac{2}{3}\frac{1}{n}h^{-1/3}S_0^{1/2}

.. math:: h_t + \left(\frac{2}{3}\frac{1}{n}h^{-1/3}S_0^{1/2}\right)h_x = 0
